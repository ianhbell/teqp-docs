\doxysubsection{Detailed Description}
A class used to evaluate mixtures using PC-\/\+SAFT model

This is the classical Gross and Sadowski model from 2001\+: \href{https://doi.org/10.1021/ie0003887}{\texttt{ https\+://doi.\+org/10.\+1021/ie0003887}}

with the errors fixed as noted in a comment\+: \href{https://doi.org/10.1021/acs.iecr.9b01515}{\texttt{ https\+://doi.\+org/10.\+1021/acs.\+iecr.\+9b01515}}
