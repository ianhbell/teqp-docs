\doxysubsection{Detailed Description}
Implementation of the polynomial extended corresponding states (p\+ECS) mixture model ~\newline
The mixture model depends on coefficients for a temperature and density polynomial. ~\newline
Only valid for binary mixtures. ~\newline
The p\+ECS reducing function follow the mathematical operations\+: ~\newline
 $F(\bar x, Y, T, \rho ) = \sum_{i=1}^{N} x_{i}^{2} Y_{\mathrm{c},i} + \sum_{i=1}^{N-1}\sum_{j=i+1}^{N} 2 x_{i} x_{j}  Y_{ij} f_{Y}(T,\rho).$ ~\newline
 $f_{T,v} = \sum_{i}^{n} \sum_{j}^{m-i} c_{ij,T,v} \tau_{\mathrm{ECS}}^{i} \delta_{\mathrm{ECS}}^{j}$ ~\newline
 $\tau_{\mathrm{ECS}} = \frac{Y_{T,ij}}{T}$ ~\newline
 $\delta_{\mathrm{ECS}} = \rho Y_{v,ij}$ ~\newline
 $Y_{T,ij} =  \sqrt{T_{\mathrm{c},i} T_{\mathrm{c},j}}$ ~\newline
 $Y_{v,ij} =  \left( \frac{1}{\rho_{\mathrm{c},i}^{1/3}} + \frac{1}{\rho_{\mathrm{c},j}^{1/3}}\right)^{3}$ ~\newline
 $c_{ij,T,v} = a_{ij,1} + a_{ij,2} x_{i} + a_{ij,3} x_{i}^{2}$.
