template<typename NumType>
class teqp::MathiasCopemanAlphaFunction< NumType >

The Mathias-Copeman alpha function used by Peng-Robinson and SRK.

\[ \alpha_i = (1+c_0x + c_1x^2 + c_2x^3)^2 \]

with

\[ x = 1-\sqrt{\frac{T}{T_{ci}}} \]
