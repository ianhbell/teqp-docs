\doxysubsection{Detailed Description}
$ \alpha^{\rm ig}= ln(\rho) + a_1 + a_2/T $

which should be compared with the original form in GERG (and REFPROP and Cool\+Prop)

$ \alpha^{\rm ig}= ln(\delta) + a_1^* + a_2^*\tau $

Note that a\+\_\+1 contains an additive factor of -\/ln(rho\+\_\+r) and a\+\_\+2 contains a multiplicative factor of Tc relative to the former
