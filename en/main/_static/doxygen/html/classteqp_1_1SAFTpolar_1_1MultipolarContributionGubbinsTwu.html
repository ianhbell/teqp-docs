Detailed Description

template<class JIntegral, class KIntegral>
class teqp::SAFTpolar::MultipolarContributionGubbinsTwu< JIntegral, KIntegral >
Template Parameters
JIntegralA type that can be indexed with a single integer n to give the J^{(n)} integral
KIntegralA type that can be indexed with a two integers a and b to give the K(a,b) integral

The flexibility was added to include J and K integrals from either Luckas et al. or Gubbins and Twu (or any others following the interface)
